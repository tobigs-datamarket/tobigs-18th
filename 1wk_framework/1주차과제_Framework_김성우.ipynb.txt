        "### Convolutional Network Class 을 정의하는 코드입니다.\n",
        "### 우선 Class 에서 pytorch 상위 클래스 Module 을 상속합니다.\n",

        "### 아래에서 kernel_size 이란 convolutional neural network 을 실행하는데 kernel 의 크기를 말합니다.\n",
        "### 5*5 의 frame 크기를 이용해 학습을 합니다. Conv2d 의 또 다른 parameter 으로 stride, padding, dilation 이 있습니다.\n",
        "\n",
        "class ConvNet(nn.Module):\n",
        "  def __init__(self): \n",
        "        super(ConvNet, self).__init__() # super 을 이용해 overloading 을 시킵니다.\n",

        "        self.drop2D = nn.Dropout2d(p=0.25, inplace=False) # drop2D 함수를 선언하는 것으로, 랜덤하게 노드 중 25% 을 제거합니다.\n",

        "        self.mp = nn.MaxPool2d(2) # mp 함수를 선언하는 것으로, 2*2 frame 에서 최대값만을 남기는 과정입니다.\n",
        "                                  # 이 과정을 거치는 이유는 유의미한 최대치인 weight 들을 많이 남기기 위함입니다.\n",
